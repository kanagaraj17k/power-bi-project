## 📌 Project Overview

This project presents an interactive **Automobile Sales Dashboard** built using **Power BI**. It analyzes sales performance data across various car models, regions, and time periods to help business stakeholders make data-driven decisions.

---

## 🎯 Problem Statement

Automobile companies often struggle to track sales performance across multiple dimensions — models, regions, dealers, and time periods. This dashboard solves that by providing a single, interactive view of all key sales metrics.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|---|---|
| Power BI Desktop | Dashboard creation & visualization |
| Microsoft Excel / CSV | Data source |
| DAX | Calculated measures & KPIs |
| Power Query | Data transformation & cleaning |

---

## 📊 Dashboard Features

- 📈 **Total Sales Trend** — Monthly and yearly sales performance
- 🚘 **Sales by Car Model** — Which models are top performers
- 🌍 **Regional Sales Breakdown** — Sales performance by geography
- 💰 **Revenue & Profit KPIs** — Key business metrics at a glance
- 🔍 **Interactive Filters** — Drill down by year, model, and region

## 💡 Key Insights

- Identified the **top 3 best-selling car models** contributing to 60%+ of total revenue
- Discovered **seasonal sales peaks** in Q1 and Q4 across all regions
- Highlighted **underperforming regions** that need targeted sales strategies
- Tracked **month-over-month growth trends** for executive reporting

---

## 🚀 How to View the Dashboard

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop** (free download at [powerbi.microsoft.com](https://powerbi.microsoft.com))
3. Interact with slicers and filters to explore the data
